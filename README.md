# Lab-3
Стек, применяемый в процессе выполнения лабораторной работы:
Docker (упаковка приложения в контейнеры).
Docker Compose (управление несколькими сервисами).
Flask (простой веб-сервер на Python).
Nginx (веб-сервер и прокси).
Redis (хранение данных в памяти).
Kubernetes-манифесты — Deployment, Service, ConfigMap.
Развертывание приложения в кластере.
Масштабирование с помощью kubectl scale.
Доступ к приложению через Service.
